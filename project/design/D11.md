# Decision 11 — The env contract: portable authored `manifest.env` + `IKIGENBA_ROOT` path composition + reduced verb set

**Decision.** A service's `manifest.env` is **input the binary reads**, not output
the binary dictates — so it is **authored, portable, and shipped**, never generated
on the box; and the on-box data paths are **composed in-binary from one box-global
root variable**, mirroring how `RESOURCE_ID`/`AUTH_SERVER` are already composed from
`IKIGENBA_DOMAIN`.

**`manifest.env` is authored and portable.** It lives committed at
`<svc>/etc/manifest.env`, carries only **portable** key/values (`APP`, `MOUNT`,
`DEFAULT`, `PORT`, `MCP`, producer retention knobs, …) and **no absolute box
paths**, ships in the bundle verbatim (D04), and is selected on-box via
`etc/current` (D02). Its readers are unchanged: the binary at `serve` (PORT and the
other knobs), the dashboard's service inventory (`inventory.Read` over
`/opt/*/etc/manifest.env`), and `bin/registry`. **Nothing generates or stamps it** —
the old "run `<bin> manifest`, stamp box paths, write `/opt/<svc>/etc/manifest.env`
every deploy" step is deleted (D10).

**The `manifest` verb is dropped.** The binary's verb set is exactly
**`serve / version / migrate / schema`**. `appkit/manifest.Emit` survives as a
**library function** (it renders a `Spec`'s portable manifest in memory) used by the
drift test below — it is no longer reachable as a CLI verb.

**Data paths compose from `IKIGENBA_ROOT`.** appkit `config` gains a
`composeDataPaths` that mirrors `composeURLs`:

```go
func composeDataPaths(getenv func(string) string, up, app string) (db, gen string) {
    root := strings.TrimSpace(getenv("IKIGENBA_ROOT"))
    if root != "" {                                              // production
        db  = filepath.Join(root, app, "state", app+".db")        // /opt/<app>/state/<app>.db
        gen = filepath.Join(root, app, "cache", app+".db.generation")
    } else {                                                     // dev fallback
        db  = "./tmp/" + app + ".db"
        gen = db + ".generation"
    }
    db  = EnvOr(getenv, up+"_DB_PATH", db)                       // explicit override wins
    gen = EnvOr(getenv, up+"_GENERATION_PATH", gen)
    return
}
```

- **Production** sets `IKIGENBA_ROOT=/opt` **once**, box-global, in the SSM-seeded
  `/etc/ikigenba/env` `EnvironmentFile` (the same file that already exports
  `IKIGENBA_DOMAIN`) — provisioned by the box's Terraform (`~/projects/metaspot`),
  not by opsctl. Every unit inherits it; the binary composes `/opt/<app>/state/…`.
- **Local dev** leaves `IKIGENBA_ROOT` unset → the `./tmp/<app>.db` default; and
  `bin/start`'s existing explicit `<APP>_DB_PATH` exports still override. **No dev
  change.**
- **opsctl's own verb runs** (migrate/backup/restore) keep injecting explicit
  `<APP>_DB_PATH`/`<APP>_GENERATION_PATH` from `Layout` (the existing `dbEnv` seam),
  so they do not depend on the box-global file being present in opsctl's environment.

**Fail-loud production guard.** Composing a wrong-but-plausible path silently (e.g.
`./tmp/<app>.db` resolved under a unit's `WorkingDirectory=/opt/<app>`) is the exact
failure to avoid. So: if `IKIGENBA_DOMAIN` is set (the production signal) **but**
`IKIGENBA_ROOT` is **not**, the binary **errors at startup** rather than booting on a
dev-default path.

**Drift safety without the verb.** A unit test asserts that each service's compiled
`Spec` defaults **byte-agree** with its committed `<svc>/etc/manifest.env` via
`manifest.Emit(spec)` — so the authored file and the binary's own notion of its
manifest cannot silently diverge once the `manifest` verb is gone.

**Rejected.**
- *Generate `manifest.env` on the box from the binary each deploy (status quo).*
  Treats config as binary output and forces an on-box generation + path-stamp step;
  the manifest is input, so it is authored and shipped. Rejected — the `manifest`
  verb and the deploy regeneration step are deleted.
- *Commit the absolute box paths into `manifest.env` (or stamp them at `ship`).* Bakes
  box-layout facts into the committed/shipped file, breaking portability and the
  byte-for-byte drift check, and duplicates layout knowledge opsctl already owns.
  Rejected for the single composed `IKIGENBA_ROOT`.
- *Derive the paths from the binary's own install location (`os.Executable`).* Works
  on the box but not for a dev build run from a checkout, and couples path policy to
  argv0; one explicit box-global var is clearer and matches the `IKIGENBA_DOMAIN`
  precedent. Rejected.
- *Default `IKIGENBA_ROOT` to `/opt` in the binary.* Erases the unset=dev /
  set=prod discriminator (dev would compose `/opt` paths); presence must signal prod,
  exactly like `IKIGENBA_DOMAIN`. Rejected for the fail-loud guard instead.

**Verification.**
- R-8DF1-W89F — every service's committed `<svc>/etc/manifest.env` is **portable**:
  it contains no absolute box path (no `/opt/…`) and no `<APP>_DB_PATH` /
  `<APP>_GENERATION_PATH` line, so the one authored file serves dev and prod
  unchanged. Substrate: a test scanning each committed `manifest.env` (shell or Go)
  asserting the absence of those tokens.
- R-8EMY-A004 — the binary's verb dispatcher exposes exactly
  `serve`/`version`/`migrate`/`schema` and has **no** `manifest` case (invoking
  `manifest` is an unknown-verb error), while `appkit/manifest.Emit` is still
  callable as a library function. Substrate: appkit verb-dispatch unit test (no
  `manifest` verb) + a direct call to `manifest.Emit`.
- R-8FUU-NRQT — `composeDataPaths` resolves correctly across the three cases: with
  `IKIGENBA_ROOT=/opt` → `/opt/<app>/state/<app>.db` + `/opt/<app>/cache/<app>.db.generation`;
  with an explicit `<APP>_DB_PATH` set → that value **overrides** the composed one;
  with neither set → the `./tmp/<app>.db` dev default. Substrate: appkit `config` unit
  test (injected `getenv`).
- R-8H2R-1JHI — startup **fails loudly** when `IKIGENBA_DOMAIN` is set but
  `IKIGENBA_ROOT` is unset (the production-misconfiguration case), instead of falling
  back to the `./tmp` dev default. Substrate: appkit `config` unit test (the error is
  returned; resolution does not silently yield a `./tmp` path).
- R-8IAN-FB87 — for **each** service, the compiled `Spec`'s default manifest
  (`manifest.Emit(spec)`) **byte-agrees** with the committed `<svc>/etc/manifest.env`
  — an edit to one that is not reflected in the other fails the test. Substrate: a Go
  unit test comparing `manifest.Emit(spec)` to the committed file bytes per service.
