# Decision 1 ‚Äî Module dependency

## Decision

`prompts/go.mod` requires the published agentkit at the release that carries the model catalog, the OpenRouter provider, and the `toolkit` subpackage:

```
require github.com/ikigenba/agentkit v0.7.0
```

There is no local `replace` for agentkit. The `appkit`, `eventplane`, and `registry` replace directives are unchanged.

The repo-root `go.work` resolves prompts' agentkit dependency from the module cache during local development, exactly as production (`GOWORK=off`) does. The import-guard tests continue to assert that no retired local-fork import path appears.

The surface prompts consumes (research ¬ß2, ¬ß5, ¬ß6):

- the advisory `catalog` package (`Lookup`, `Resolve`, `ListByProvider`, `Entry`, `ReasoningSpec`) ‚Äî research ¬ß5;
- typed-credential provider constructors (`New(cred Credential, opts ...Option)` on every provider sub-package);
- the `github.com/ikigenba/agentkit/openrouter` provider;
- consumer-owned cost resolution via `Conversation.Pricing`;
- the `github.com/ikigenba/agentkit/toolkit` subpackage (new in v0.7.0) ‚Äî the six standard coding tools (`Bash`, `Read`, `Write`, `Edit`, `Glob`, `Grep`) as ready-made `agentkit.Tool` values, consumed by D5 in place of prompts' own implementations ‚Äî research ¬ß6.

## Rejected

**Stay on v0.6.0 and keep the local tool implementations** ‚Äî rejected: v0.7.0's only delta is the `toolkit` subpackage, which replaces ~530 lines of tool code prompts otherwise owns and maintains (timeouts, output caps, ambiguous-edit refusal, `**` globbing all come for free). The whole point of the bump is to stop re-inventing the common tools per agent.

**Vendor a model list inside prompts** ‚Äî rejected: the catalog is agentkit's authoritative, versioned statement of the supported provider/model/effort combinations; a private copy would drift.

## Verification

Pure structural decision ‚Äî enables D2‚ÄìD5, D7, D9. No behavior ids of its own; correctness is implied when the suite compiles against v0.7.0 and the behavioral ids of the dependent decisions pass.
